# 비트코인 트레이딩 프로그램

## 주요 기능

- 거래를 실행할 거래소를 선택한다.
- 설정한 가격, 수량으로 자동으로 거래를 실행한다.
- 거래 실행 후 설정한 TL/SL가 있다면 자동으로 설정한다.
- 거래 실행 후 설정한 Close가 있다면 자동으로 설정한다.
- 거래 히스토리를 확인한다.
- 모든 거래는 비트코인의 선물 차트를 사용한다.

## 기술스택

- Next.js
- ccxt
- Tailwind CSS
- TypeScript
- Supabase

## 기능 1. 로그인 및 설정

- Supabase Auth를 사용하여 로그인한다.
- 로그인 후 메인 페이지로 이동한다.
- 로그인 후 설정 버튼으로 설정 페이지로 이동한다.
- 설정 페이지에서 거래소 별 거래소 API 키를 입력한다.

## 기능 2. 거래

### 1. 진입가 설정

- 거래할 거래소를 선택한다.
- 거래소를 선택하면 해당 거래소의 잔액을 표시한다.
- 레버리지 (기본 x10)와 롱/숏을 선택한다.
- 진입가격을 입력한다.

### 2. TP/SL 설정

- 진입가격을 입력하면 TP/SL 선택, 입력창을 표시한다
- TP/SL 설정 시 가격을 직접 입력하거나, 진입가 대비 %로 입력할 수 있다.
- %는 3, 5, 10% 선택 버튼이 있고 직접 입력도 가능하도록 한다.
- TP/SL 설정 후 거래가 체결되면 설정대로 TP/SL을 설정한다.

### 3. Close 설정

- Close 설정 시 가격을 직접 입력하거나, 진입가 대비 %로 입력할 수 있다.
- %는 3, 5, 10% 선택 버튼이 있고 직접 입력도 가능하도록 한다.
- Close 설정 후 거래가 체결되면 설정대로 Close를 설정한다.

### 4. 거래 실행

- 거래 설정이 완료되면 거래 실행 버튼을 표시한다.
- 거래 실행 버튼을 클릭하면 거래를 실행한다.
- 거래 실행 후 거래 히스토리에 거래를 추가한다.

## 기능 3. 거래 히스토리

- 거래 히스토리를 표시한다.
- 거래 히스토리는 거래소 별로 필터링할 수 있다.
- 거래 히스토리는 날짜 별로 필터링할 수 있다.
- 거래 히스토리는 롱/숏 별로 필터링할 수 있다.
- 히스토리에는 거래소, 진입가, 종료가, 수량, PnL, 수익률, 레버리지, 날짜, 롱/숏을 표시한다.
